# ClearMoney 💎

> Personal financial clarity for Indian millennials and Gen Z.

ClearMoney is a client-side personal finance web app that parses your bank statement or UPI SMSes, computes a Financial Health Score, and surfaces AI-powered insights — all without your data ever leaving your device.

Built as a PM portfolio project, designed and engineered to production quality.

## Features

| Screen | What it does |
|---|---|
| **Onboarding** | 3-step wizard — financial profile → upload bank statement PDF or paste UPI SMSes → animated processing |
| **Dashboard** | Financial Health Score (0–100), 4 sub-scores, AI insight cards, spending breakdown, quick actions |
| **Insights** | 8 personalised insight cards with category filters and expandable "Why does this matter?" explanations |
| **Goal Simulator** | Live compound interest calculator — see exactly what you need to save/month to hit any goal |
| **Education Hub** | 6 personalised long-form articles built around your actual numbers |

### Key behaviours
- **PDF parsing** — extracts transactions from HDFC, ICICI, SBI, Axis, Kotak statements using position-based row detection via pdf.js
- **SMS parsing** — parses all major Indian UPI SMS formats
- **Real data always** — insights and scores always reflect your uploaded data, never hardcoded placeholders
- **AI insights** — optionally provide a Claude API key for LLM-generated insights; falls back to dynamically computed local insights if no key is provided
- **Zero data storage** — everything runs in the browser; nothing is sent to any server except the optional Claude API call

## Using Your Own Data

**PDF Bank Statement**
1. Log in to your bank's mobile/web app
2. Download the last 3 months as PDF
3. Upload on the Onboarding screen
4. Works with HDFC, ICICI, SBI, Axis, Kotak and most other Indian banks

**UPI SMSes**
1. Open your SMS inbox
2. Copy all transaction messages
3. Paste into the SMS text area on the Onboarding screen

**AI Insights (optional)**
1. Get a Claude API key from [console.anthropic.com](https://console.anthropic.com)
2. Paste it in the "Claude API Key" field on Onboarding Step 2
3. Insights will be generated by Claude based on your actual numbers

Without a key, insights are still fully personalised — computed locally from your transaction data.

## Financial Health Score

Weighted composite of four sub-scores:

| Sub-score | Weight | Target | Formula |
|---|---|---|---|
| Savings Rate | 30% | ≥ 20% of income | `(income − expenses) / income` |
| Emergency Fund | 30% | 3 months of expenses | `savings balance / monthly expenses` |
| Investment Exposure | 25% | ≥ 10% of income | `monthly SIP / income` |
| Debt Load | 15% | < 40% of income | `monthly EMI / income` (inverted) |
